# Product Submission Guide

Use this guide to submit your product to the community directory in Product.md.

## How To Submit

1. Fork this repository.
2. Open Product.md.
3. Copy the row template below.
4. Add your row at the end of the tbody section under Product Submissions, below the previous row.
5. Verify all links are public and working.
6. Open a pull request with title: Add <Your Product Name> to Product.md.

## Submission Rules

- One product per entry.
- No duplicate products.
- Use HTTPS links when possible.
- Keep description neutral and concise.
- No referral links, tracking links, or promotional spam.
- Follow the exact template fields.

## Template

```html
<tr>
	<td>Your Product Name</td>
	<td>SaaS / AI / Developer Tools / Marketing / etc.</td>
	<td>What your product does in one line.</td>
	<td>Who is this for?</td>
	<td>Free / Freemium / Paid / One-time</td>
	<td><a href="https://your-product-url.com">your-product-url.com</a></td>
</tr>
```

## Pull Request Checklist

- I checked that this product is not already listed.
- I used the row template with all required columns.
- I verified all links are valid.
- I kept the one-line description concise.
